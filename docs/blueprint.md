# **App Name**: Paarsh Infotech Hub

## Core Features:

- Responsive Design: Responsive layout adapting to different screen sizes using Tailwind CSS.
- Main Navigation: Navigation menu with links to Home, About, Services, Careers, Media, and Contact pages.
- Contact Form: Contact form for user inquiries, collecting Name, Email, and Message.
- Quote Form: Quote request form to gather requirements for potential clients.
- Stats Display: Display company stats such as Happy Clients, Projects Completed, etc., using animated counters.
- Smart Reply Tool: AI-powered tool that assists with generating personalized follow-up email content based on the client's initial message and selected service(s), streamlining communication.
- 404 Error Handling: Basic error page to be displayed if no route is found

## Style Guidelines:

- Primary color: Deep Indigo (#4F3A65) to convey professionalism and stability.
- Background color: Very light gray (#F5F5F7) to provide a clean, neutral backdrop.
- Accent color: Muted Blue (#7CB9E8) for highlighting calls to action and important information.
- Body and headline font: 'Inter' sans-serif for a modern, clean and neutral design.
- Grid layout for services and clients sections.
- Full-width sections with adequate padding for comfortable reading.
- Smooth scrolling for navigation within the page.
